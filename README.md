FeedMyTeacherBack
=================
A symfony based school project. 

### Who's behind  : 
@JulesAaelio and @AlexandraRamadour

### What's the point ? : 
Provide a tool that allow student to give feedback on their classes / teachers. 

###How to install :
```
git clone git@github.com:JulesAaelio/feedMyTeacherBack.git
vagrant up
vagrant ssh
make fixtures
make start 
```
